Here is a compact starter set you can upload as separate knowledge documents.

`faq-sample.md`
```md
# Customer FAQ

## Password reset
If you forgot your password, use the password reset link on the login page.
A reset email will be sent to the address on your account.

## Order tracking
Go to My Orders, open the order, and check the shipping status and tracking number.

## Contact support
Use the chat widget and choose Talk to an agent for human support.
```

`refund-policy.md`
```md
# Refund Policy

## Return window
Most items can be returned within 30 days of delivery.

## Refund timing
Approved refunds are typically processed within 5 to 10 business days.

## Conditions
Refunds may be delayed if the item is missing, damaged, or outside the return window.
```

`shipping-help.md`
```md
# Shipping Help

## Missing tracking
If your order has not received a tracking number yet, allow 24 to 48 hours after shipment.

## Delayed delivery
If a package is delayed, check the carrier status first before contacting support.

## Address issues
If the shipping address is incorrect, contact support immediately.
```

`agent-notes.md`
```md
# Agent Notes

## Escalation
Escalate when the customer requests a supervisor, reports repeated failures, or disputes a refund.

## Order lookup
Order lookup requires the order number and customer identity.

## Refunds
Check eligibility before creating a refund request.
```

`returns-manual.md`
```md
# Returns Manual

## Eligibility
Items must be in returnable condition and inside the return window.

## Workflow
1. Verify the order.
2. Check return eligibility.
3. Create the return or refund request.
4. Confirm the next steps to the customer.

## Exceptions
Escalate damaged, incomplete, or policy-exception cases.
```

Suggested `sourceType` values:
- `FAQ` for `faq-sample.md`
- `POLICY` for `refund-policy.md`
- `SUPPORT_ARTICLE` for `shipping-help.md`
- `MANUAL` for `agent-notes.md`
- `MANUAL` or `POLICY` for `returns-manual.md`
