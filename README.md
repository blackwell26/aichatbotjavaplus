# aichatbotjava

AI-powered customer service chatbot for an e-commerce platform.

## Build

```powershell
mvn test
mvn package
```

## Run

```powershell
mvn spring-boot:run
```

## Project Layout

- `src/main/java/com/company/chatbot` for the Spring Boot backend.
- `src/main/resources` for Spring profiles and application configuration.
- `src/test/java/com/company/chatbot` for automated tests.
- `specs` for requirements, technical design, and implementation task tracking.
